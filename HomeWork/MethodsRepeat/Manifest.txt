В этом файле буду помечать то, что упустил, при первом изучении массивов и их методов.

"Напоминания":
1) fruits[fruits.length] = 'Груша' === fruits.push = 'Груша' - добавляет в конец массива новый элемент.

2) Не используем "for ... in ..." с массивами. Это в 10-100 раз дольше, чем "for ... of ...".

3) Свойство length можно перезаписать вручную. Получим массив с "дырами" или "обрезанный" массив.

4) let arr = new Array (число) используется для создания пустого массива указанной длины. Используется очень редко.

5) Разные (не скопированные) массивы не сравниваются с помощью "==". Используйте циклы или методы итерации.

6) Элемент из конца массива можно получить с помощью array.at (-i).

7) arr.splice возвращает массив из удалённых элементов. 

8) arr.concat не умеет "по дефолту" добавлять элементы объектов. Если надо скопировать массивоподобный объект - 
используем свойство [Symbol.isConcatSpreadable]: true.

9) Результат функции внутри arr.forEach (если она что-то возвращает) игнорируется.
